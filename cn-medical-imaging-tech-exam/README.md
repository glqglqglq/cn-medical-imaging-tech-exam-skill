# cn-medical-imaging-tech-exam

这是一个可上传到支持 Skills 环境中的纯 Skill 目录。

## 使用方式

上传或复制本目录：

```text
cn-medical-imaging-tech-exam/
```

然后提出请求，例如：

```text
请使用 cn-medical-imaging-tech-exam skill，根据我上传的知识点，生成 40 道中国医学影像技术职称考试原创单项选择题。
```

## 目录说明

- `SKILL.md`：Skill 的核心说明文件；
- `references/`：考试范围、命题规则、输出格式、质量检查和知识点资料；
- `examples/`：常见请求示例。

## 重要说明

本 Skill 没有微调任何 LLM，不包含模型权重或 API 配置。它通过规则、知识点和示例引导通用 LLM 完成医学影像技术职称考试相关任务。

本 Skill 中的知识点是可编辑参考脚手架，可按使用者自己的资料进行删减、修改和扩展。
