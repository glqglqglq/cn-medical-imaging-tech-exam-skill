# cn-medical-imaging-tech-exam-skill
# cn-medical-imaging-tech-exam-skill  面向 **中国医学影像技术职称考试** 的 Codex-ready Skill。  本 Skill 专注于中国医学影像技术专业技术人员职称考试的复习与命题场景，可用于生成、审核和重构原创练习题。它不是通用医学题库，也不面向临床执业医师、护士资格、放射医师或其他医学考试。  支持的题型包括：  - 单项最佳选择题 - A2 共用题干题 - B1 共用选项题  本 Skill 强调医学影像技术岗位能力，包括检查准备、体位摆放、扫描参数、图像质量控制、设备使用、辐射防护、造影剂安全和检查流程管理等内容。
