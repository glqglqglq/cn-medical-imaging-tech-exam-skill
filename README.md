## 🌟 项目定位

`cn-medical-imaging-tech-exam-skill` 是一个专注于 **中国医学影像技术职称考试** 的开放式 Skill，用于辅助生成、审核和重构原创练习题。

本项目不是通用医学题库，不面向临床执业医师、护士资格、放射医师或其他医学考试。它的设计重点始终聚焦于 **医学影像技术岗位能力** 与 **中国职称考试复习场景**，强调基础知识、相关专业知识、专业知识与专业实践能力的系统覆盖。

本项目也不是微调模型。它不包含模型权重、训练数据、推理服务或 API 绑定，而是通过 `SKILL.md`、`references/` 和 `examples/` 引导通用 LLM 在中国医学影像技术职称考试场景下进行结构化命题。

---

## 📚 知识点来源说明

本项目中的知识点文件主要来自公开资料整理与爬虫提取后的结构化归纳，围绕中国医学影像技术职称考试常见复习模块进行组织。

这些文件 **不是官方考试大纲**，**不是官方题库**，也不是对真实考试题的还原。它们更适合作为可编辑的知识点脚手架：

- 使用者可以根据自己的考试级别、地区要求、教材版本或教学目标进行删减；
- 可以补充自己的复习笔记、讲义、教材章节提纲或培训材料；
- 可以重新分类、合并、去重和修订知识点；
- 可以根据不同需求调整四大模块的题量比例和命题重点。

本项目的目标不是提供固定题库，而是提供一个可复用、可修改、可迁移的医学影像技术职称考试 Skill 框架。

---

## 🧠 关于模型与微调

本项目 **没有对任何大语言模型进行微调**。

它不是 fine-tuned model，也不包含模型权重、训练脚本、私有数据集或模型服务。Skill 的行为来自：

- `SKILL.md` 中的任务说明；
- `references/` 中的考试范围、题型规则和知识点；
- `examples/` 中的示例请求；
- 用户上传或自行维护的补充资料。

因此，本 Skill 可以被复制到不同支持 Skills 的环境中使用，也可以根据使用者自己的模型、资料和输出格式进行修改。

## 🏛️ 项目价值

中国医学影像技术职称考试具有明显的专业岗位属性。它既考查医学基础与影像学基础，也强调检查技术、设备原理、图像质量控制、辐射防护、患者流程与实践能力。

通用医学题目生成往往容易偏向疾病诊断、治疗决策或临床医师视角，而本 Skill 通过明确的题型规则、考试范围边界和质量检查清单，将生成方向约束在医学影像技术人员的岗位能力上。

因此，本项目更适合用于医学影像技术职称考试复习、教学辅助、培训命题和题目质量审核。

## 📥 支持的输入资料

本 Skill 适合处理以下类型的输入：

- 爬虫整理的关键知识点；
- 自有复习笔记；
- 教材章节提纲；
- 教学讲义；
- 考点清单；
- 已生成题目的质量审核文本；
- A1/A2/B1 题型模板。

不建议直接输入：

- 真实考试原题；
- 受版权保护的完整题库；
- 未经授权的大段教材原文；
- 需要模型还原、仿写或规避版权保护的资料。

## 🧩 支持题型

### ✅ 单项最佳选择题

适用于基础概念、设备原理、检查流程、参数选择、图像质量控制、辐射防护等知识点的考查。

### 🧾 A2 共用题干题

适用于医学影像技术岗位中的场景化考查，例如检查准备、体位摆放、扫描参数选择、图像质量分析、患者安全处理等。

### 🔗 B1 共用选项题

适用于配伍选择、概念对应、设备参数、检查方法、影像质量因素、辐射防护原则等内容。

## 🩻 聚焦的医学影像技术能力

本 Skill 强调医学影像技术人员在实际岗位中的核心能力，主要包括：

- X 线摄影技术；
- CT 检查技术；
- MRI 检查技术；
- DSA 与介入相关技术基础；
- 超声与核医学基础；
- 图像质量控制；
- 影像设备基础与日常维护；
- 辐射防护与安全管理；
- 造影剂使用安全；
- 患者检查准备与流程管理；
- 检查风险识别与应急处理。

## 🚀 Quick Start

1. 克隆或下载本仓库。
2. 找到 Skill 目录：

   ```text
   cn-medical-imaging-tech-exam/
   ```

3. 将该目录上传或复制到支持 Skills 的环境中。
4. 使用类似请求：

   ```text
   请使用 cn-medical-imaging-tech-exam skill，根据我上传的知识点，生成 40 道中国医学影像技术职称考试原创单项选择题。
   ```

5. 对生成题目进行人工审核和修改。

---

## 🔁 推荐工作流

1. **准备知识点**  
   使用爬虫、人工整理或自有资料形成知识点清单。

2. **清洗知识点**  
   删除重复、过时、明显错误或超纲内容。

3. **按模块归类**  
   建议归入基础知识、相关专业知识、专业知识、专业实践能力四个模块。

4. **调用 Skill 生成题目**  
   根据目标题型生成单项选择题、A2 共用题干题或 B1 共用选项题。

5. **进行质量审核**  
   检查是否存在超纲、歧义、答案不唯一、干扰项过弱、临床医师视角过重等问题。

6. **人工复核**  
   对医学表述、考试适配性和知识点准确性进行最终确认。

## 🛠️ 如何自定义知识点

使用者可以根据需要修改 `cn-medical-imaging-tech-exam/references/knowledge-points/` 中的知识点文件：

- 删除不适合自己考试级别的内容；
- 补充本地区、本机构或个人复习资料中的重点；
- 调整四大模块的知识点比例；
- 增加新的题型模板或输出格式；
- 将爬虫整理出的知识点进一步清洗、去重、分类。

建议修改时保留四个一级模块：

1. 基础知识；
2. 相关专业知识；
3. 专业知识；
4. 专业实践能力。
