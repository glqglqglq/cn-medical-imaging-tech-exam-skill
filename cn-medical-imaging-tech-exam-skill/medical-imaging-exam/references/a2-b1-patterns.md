# A2 and B1 Question Patterns

## A2 shared-stem questions

A2 shared-stem groups should include:

- One shared clinical or examination scenario
- Multiple subquestions based on the same scenario
- Independent A-E options for each subquestion
- One best answer per subquestion

The shared stem should emphasize imaging technology work, such as:

- Examination preparation
- Positioning
- Parameter choice
- Sequence or protocol selection
- Image quality problem analysis
- Patient safety
- Radiation protection
- Contrast media workflow

Avoid making the group depend mainly on final clinical diagnosis or treatment decision.

## B1 shared-option questions

B1 shared-option groups should include:

- One theme or common topic
- Five shared options A-E
- Multiple subquestions that each match one option
- One best answer per subquestion

Good B1 option sets are parallel and confusable, such as:

- Imaging positions
- CT window settings
- MRI sequences
- Image artifacts
- Radiation protection principles
- Quality control terms
- Contrast media adverse reaction handling steps
- Imaging equipment components

The same option may be used more than once only if the user allows it. Otherwise, try to distribute answers when reasonable.
