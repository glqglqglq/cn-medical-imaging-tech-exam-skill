# Exam Scope / 考试范围

本 Skill 面向 **中国医学影像技术职称考试**，建议采用以下四大模块组织知识点与题目。

## 1. 基础知识

- 人体解剖学基础；
- 生理学基础；
- 病理学基础；
- X 线物理基础；
- 医学影像成像基础；
- 医学影像相关基础概念。

## 2. 相关专业知识

- 影像设备基础；
- X 线机结构与控制；
- CR、DR、CT、MRI、DSA 基础；
- PACS 与图像显示；
- 医学影像信息系统；
- 辐射剂量与防护基础。

## 3. 专业知识

- X 线摄影技术；
- CT 检查技术；
- MRI 检查技术；
- DSA 与介入相关技术基础；
- 图像质量评价；
- 数字图像处理；
- 对比剂与检查安全；
- 影像设备参数与图像质量关系。

## 4. 专业实践能力

- 检查准备；
- 体位摆放；
- 扫描基线选择；
- 扫描参数选择；
- 图像质量分析；
- 患者安全；
- 辐射防护；
- 造影剂风险识别；
- 流程管理；
- 应急处理。

## Scope Boundary / 范围边界

题目应突出医学影像技术岗位任务，避免过度考查：

- 临床诊断结论；
- 治疗方案选择；
- 医师处方决策；
- 与医学影像技术职称考试无关的泛医学内容。
